# Compilation PDF Notes

Use this folder as your personal organizer for source files.

## Suggested workflow
1. Keep your original files here while drafting.
2. Copy the final files to public/portfolio/ so Next.js can serve them.
3. In page configs (Midterm/Final), set src to paths like:
   - /portfolio/midterm/midterm-quiz-1.pdf
   - /portfolio/final/final-project.pdf
   - /portfolio/midterm/activity-1-output.png

## Supported previews in the current cards
- PDF files via iframe src
- Images via direct image URL if you switch card component later

## Naming tip
Use lowercase and dashes for file names so links stay clean.
